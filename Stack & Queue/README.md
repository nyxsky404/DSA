## Complexity Analysis

### Stack Operations

- Push: O(1)
- Pop: O(1)
- Peek: O(1)
- Space: O(n)

### Queue Operations

- Enqueue: O(1)
- Dequeue: O(1)
- Front: O(1)
- Space: O(n)
